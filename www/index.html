Bienvenue sur WebServ !

Ce serveur web supporte les scripts Python CGI dynamiques. Testez les différentes méthodes HTTP :

  • Accueil CGI (GET) → Test GET, formulaire, navigation
  • Formulaire POST → Test POST, affiche les données envoyées
  • Suppression DELETE → Démo suppression (simulée)
  • Infos CGI → Variables d'environnement et headers
  • Liste & Test CGI → Page de test GET/POST, navigation
